### 2. Google Search Console (15 minutes) - CRITICAL
**Why:** Tells Google about your site and tracks rankings

**Steps:**
1. Go to: https://search.google.com/search-console
2. Click "Start now"
3. Choose "URL prefix" method
4. Enter: https://wellingtontimes.co.nz
5. Verify ownership:
   - Download HTML file
   - Upload to your website root
   - Or use DNS verification
6. Once verified:
   - Submit sitemap: https://wellingtontimes.co.nz/sitemap.xml
   - Request indexing for homepage
   - Request indexing for top 5 articles

**Result:** Google starts tracking your performance within 24-48 hours

---

### 3. Google Analytics (15 minutes)
**Why:** Track your traffic and understand what's working

**Steps:**
1. Go to: https://analytics.google.com
2. Create account: "Wellington Times"
3. Create property: "wellingtontimes.co.nz"
4. Get tracking ID (starts with G-)
5. Update index.html:
   - Find: `G-XXXXXXXXXX`
   - Replace with your real tracking ID
6. Set up goals:
   - Newsletter signup
   - Article read (time on page >1 min)
   - Social share clicks

**Result:** Start seeing visitor data within hours

## üÜò TROUBLESHOOTING

### "I'm not showing up in Google"
- Wait 48-72 hours after Search Console verification
- Make sure robots.txt isn't blocking Google
- Request indexing in Search Console
- Check if pages are mobile-friendly

### "Google Business Profile not showing"
- Make sure you verified it
- Add more photos (minimum 10)
- Get 5 reviews
- Post weekly updates
- Use precise location

### "Not ranking for keywords"
- Are you using those keywords in titles?
- Do you have local content?
- Is your site fast? (Test with PageSpeed Insights)
- Do you have backlinks? (Even 5 helps)
